== Description ==

Add ticket sales and a calendar to your Wordpress blog by integrating with BuyPlayTix.

== Installation ==

Please follow the standard installation procedure for WordPress plugins.

== Frequently Asked Questions ==

= Do I need an account to use this plugin? =

Yes. You will need a BuyPlayTix account.
